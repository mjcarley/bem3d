\subsection{Detailed Description}
B\+E\+M3D can handle a wide range of elements and some of them are built in to the main library. The B\+E\+M\+Element type contains the geometric and other information needed to perform calculations; the \doxyref{B\+E\+M3\+D\+Element\+Build\+Func}{p.}{structBEM3DElementBuildFunc} type defines a function which takes as input the vertices of an element and returns the new element. In each case, the inputs are an array of Gts\+Edge\textquotesingle{}s and an array of Gts\+Vertex\textquotesingle{}s which define the boundary of the element, with the exception of \doxyref{bem3d\+\_\+element\+\_\+build\+\_\+t3}{p.}{group__elements_ga0d3d9bc819bb8643592b2d0cb320a620} and \doxyref{bem3d\+\_\+element\+\_\+build\+\_\+q2}{p.}{group__elements_ga5a41eaafa4f26fae1f1112b4dfdc5b4d} which include an interior point in the centre of the element.
