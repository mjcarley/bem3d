bem3d2pos: convert BEM3D meshes to GMSH .pos format

This code is now superseded by ::bem3d2msh

The GMSH pos format is deprecated, since visualization is now done with mesh files, but it is still sufficient for visualization of data from BEM3D calculations.

The most basic invocation of bem3d2pos is 

bem3d2pos -i input.bem -o output.pos

which generates a visualization file output.pos which can be viewed with 

gmsh output.pos

An extra option allows you to visualize the surface with solution information added: 

bem3d2pos -i input.bem -d input.dat -f 4 -o output.pos

will generate a file output.pos which contains the geometry of input.bem with the nodes coloured according to the value of field 4 of the data block in input.dat (the field option defaults to 0).

The edge data for the input file can also be visualized, for example, to select the trailing edge in a bem3d-aero calculation: 

bem3d2pos -i input.bem -e input-0.edg -e input-1.edg -o output.pos

generates a .pos file of input.bem with additional entities for the edge files input-0.edg and input-1.edg.
