Detailed Description

BEM3D codes can be configured at run-time using glib key files and the following functions.

gint bem3d_configuration_set ( BEM3DConfiguration *  c,

Set fields of a BEM3DConfiguration from a key file, overwriting previous settings. Settings are read from the "BEM3D" group of the file, to allow codes to keep all their settings in one place. Values not in the "BEM3D" group are ignored.
