Detailed Description

The Fast Multipole Method (FMM) is a standard technique for accelerating BEM calculations. BEM3D includes an interface to FMM solvers developed by other groups, identified by BEM3DFastMultipole

BEM3DFMMMatrix* bem3d_fmm_matrix_new ( BEM3DFastMultipole  solver,
BEM3DFastMultipoleProblem  problem,
BEM3DMeshSkeleton *  skel,
BEM3DConfiguration *  config,
BEM3DParameters *  param,
gdouble  r,
BEM3DWorkspace *  work 
)

Generate a new BEM3DFMMMatrix which can be used in matrix-free calculations. The underlying calculation method uses a set of point sources to represent the mesh, in skel. This is then `corrected' by subtracting sources which lie within distance r of a collocation point and adding the integrated contribution from those elements calculated using the quadrature rules set in the configuration config.

Parameters
solverthe FMM solver to be used;
problemproblem to be solved by FMM solver (e.g. Helmholtz or Laplace)
skela BEM3DMeshSkeleton for the source points which will be passed to the FMM solver;
configconfiguration for the problem (including quadrature rules for correction calculation);
paramparameters for the physics;
rradius for elements to included in correction calculation;
worka BEM3DWorkspace.
Returns
BEM3D_SUCCESS on success.
