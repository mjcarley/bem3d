BEM3DGreensFunc

gint BEM3DGreensFunc(GtsPoint *x, GtsPoint *y, GtsVector n, BEM3DParameters *p, gdouble *G, gdouble *dGdn)

Green's function for various problems. The function should return 0 on success and fill G with the Green's function and dGdn with its normal derivative, or their equivalents for a particular problem.

Parameters
xfield point
ysource point
nnormal at source point
paramparameters to use in Green's function
GGreen's function
dGdnnormal derivative of Green's function
