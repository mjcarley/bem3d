#define bem3d_parameters_amplitude_real (   p)

Real part of amplitude in a BEM3DParameters struct. The real and imaginary parts are aligned so that they can be extracted together and cast, if necessary, to a gsl_complex type. That is: 

&(bem3d_parameters_amplitude_real(p))

is a pointer to an array of type gdouble of length two, containing the real and imaginary parts of the magnitude.

BEM3DGreensFunc

gint BEM3DGreensFunc(GtsPoint *x, GtsPoint *y, GtsVector n, BEM3DParameters *p, GArray *G, GArray *dGdn)

Green's function for various problems. The function should return 0 on success and fill G with the Green's function and dGdn with its normal derivative, or their equivalents for a particular problem.

Parameters
xfield point
ysource point
nnormal at source point
paramparameters to use in Green's function
GGreen's function
dGdnnormal derivative of Green's function
