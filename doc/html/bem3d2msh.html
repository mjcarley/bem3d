bem3d2msh: convert BEM3D meshes to GMSH .msh format

The most basic invocation of bem3d2msh is 

bem3d2msh -i input.bem -o output.msh

which generates a mesh file output.msh which can be viewed with 

gmsh output.msh

An extra option allows you to visualize the surface with solution information added: 

bem3d2msh -i input.bem -d input.dat -f 4 -o output.msh

will generate a file output.msh which contains the geometry of input.bem with the nodes coloured according to the value of field 4 of the data block in input.dat (the field option defaults to 0).

The main purpose of the program is visualization of geometries and data, rather than meshing.
