Apply a named reduction operation to a BEM3DMeshData on a BEM3DMesh. Reduction operations can be listed by calling bem3d_reduction_func_list.

Parameters

ma BEM3DMesh
fa BEM3DMeshData containing data for m
idataarray containing integer data on exit
ninumber of entries in idata on exit
ddataarray containing double precision data on exit
ndnumber of entries in ddata on exit
dataignored
Returns
0 on success

gint bem3d_reduction_func_list ( FILE *  output,
gchar *  format,
gboolean  describe 
)

Write a list of available reduction operations.

Parameters
outputa FILE stream to write information to;
formata `printf' format string for the data;
describeif TRUE, also write the short description of the reduction operation

The format string should be appropriate to the setting of describe. If describe is FALSE, format should contain one entry of `s' (for the name of the operation); if TRUE, format should contain `s' twice (once for the name of the operation and once for its description).

Returns
0 on success
