Typedef Documentation

BEM3DElement

Data type for BEM3D elements, containing the data for geometry and collocation points.

The BEM3DElement type should usually be accessed only through the provided functions and macros. In order to implement new elements, you should look up the BEM3DElementBuildFunc type.

The basic class for a BEM3DElement
