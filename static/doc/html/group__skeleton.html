Detailed Description

BEM3D skeletons are representations of meshes as arrays of point sources, mainly intended for use in interfacing to third-party libraries such as Fast Multipole Methods which take as input a set of elementary sources.

Initialize a BEM3DMeshSkeleton which has been allocated with bem3d_mesh_skeleton_new. You should be careful to use a BEM3DQuadratureRule which guarantees source points which are distinct from the collocation points. For example, a three-point Gaussian quadrature with nodes at the midpoints of triangle edges will generate source points which coincide with the mid-edge nodes of a second-order triangular element.
