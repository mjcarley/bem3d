Perform a matrix multiplication using a ::BEM3DFMMMatrix, .

Parameters:
fa ::BEM3DFMMMatrix;
ldatadata to be passed to ::bem3d_fmm_matrix_lookup_func(f);
xlfunc::GmcLookupFunc for the left hand side vector, or NULL;
xdatadata for xlfunc;
ylfunc::GmcLookupFunc for the right hand side vector, or NULL;
ydatadata for ylfunc.

If xlfunc (ylfunc) is NULL, then xdata (ydata) may not be NULL and will be treated as a gdouble array and a built-in lookup function will be used. Also, note that ylfunc is used to insert the result of multiplication in its vector, so it must return a pointer to the true location for the result and not to some intermediate location or buffer.

Returns:
::BEM3D_SUCCESS on success.
